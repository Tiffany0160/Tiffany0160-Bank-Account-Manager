# Bank Account Manager

A simple Java console-based banking system that allows users to check their balance, deposit money, withdraw money, and exit the program.

## Features
- Check account balance
- Deposit money
- Withdraw money
- Prevent withdrawals when funds are insufficient

## Technologies Used
- Java
- Scanner class
- Conditional statements and loops

## How to Run
1. Open the project in a Java IDE
2. Run the `BankAccountManager.java` file
3. Use the menu to manage the bank account
